# PWA Icons

This directory should contain the following PWA icons:

## Required Icons

- `icon-192.png` - 192x192 pixels
- `icon-512.png` - 512x512 pixels
- `favicon.ico` - 16x16, 32x32 pixels
- `apple-touch-icon.png` - 180x180 pixels

## How to Create Icons

1. Create a square logo/design for your app
2. Use a tool like:
   - https://realfavicongenerator.net/ (recommended)
   - https://www.favicon-generator.org/
   - https://www.pwabuilder.com/imageGenerator
3. Upload your design and download all required sizes
4. Place the generated icons in this `public/` directory

## Icon Design Guidelines

- Use a simple, recognizable design
- Ensure good contrast on both light and dark backgrounds
- Use the app's primary color (#3b82f6)
- Keep the design clean and minimal
- Test on different backgrounds

## Current Status

The PWA is configured to use these icons, but they need to be created manually.
The app will still work without them, but won't have proper icons when installed.
