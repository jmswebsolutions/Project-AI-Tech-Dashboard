# AI & Tech Dashboard

A modern dashboard focused on Artificial Intelligence and Technology news.

## 🚀 About the Project

AI & Tech Dashboard is a web application built to centralize technology and AI-related information in a clean and intuitive interface.

The project aims to provide users with:

* AI news
* Technology news
* Search functionality
* Category filters
* Favorites system
* Data visualization and analytics

## 🛠 Technologies

* React
* TypeScript
* Vite
* Tailwind CSS
* Axios
* React Query
* Recharts

## 📌 Planned Features

* [ ] Display technology news
* [ ] Search news articles
